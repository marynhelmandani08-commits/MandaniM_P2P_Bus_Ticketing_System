#  SakayOne P2P Bus Ticketing System

##  Overview
The SakayOne P2P Bus Ticketing System is a Java-based desktop application designed to manage bus ticketing operations efficiently. It uses a MySQL database to store ticket records, user accounts, and transaction data.

This system replaces manual ticketing by providing a fast, organized, and user-friendly interface for both administrators and cashiers.

---

##  Features

- *Role-Based Login System*
  - Admin and Cashier access control

- *Admin Dashboard*
  - Manage system data
  - Monitor ticket sales and reports

- *Cashier Panel*
  - Book tickets
  - Enter passenger details
  - Generate receipts

- *Ticket Management*
  - Store and retrieve ticket records
  - Automatic timestamping

- *Database Integration*
  - MySQL backend for data storage

---

##  Prerequisites

Before running the system, make sure you have:

- *JDK*: Version 8 or higher
- *Database*: MySQL Server
- *IDE*: NetBeans (recommended)
- *Libraries*:
  - MySQL Connector (JDBC)
